# Chat-Lib
A chat library to control & style a message for players

# Maven
```xml
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository>

<dependency>
    <groupId>com.slyvr.libs</groupId>
    <artifactId>chat-lib</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
```
